📋 Términos de Uso

DULA Content Creator es una aplicación web de uso personal y profesional para la creación de contenido asistida por inteligencia artificial. Al usar esta aplicación, aceptas utilizar el contenido generado de forma ética, legal y respetando los derechos de autor de terceros.

🔒 Política de Privacidad

Esta aplicación no recopila ni transmite datos personales a servidores externos. Toda la información (notas, prompts, configuraciones) se almacena localmente en tu dispositivo mediante IndexedDB. Las API keys que introduzcas se guardan únicamente en tu navegador local.

🤖 Uso de IA

El contenido generado por inteligencia artificial es de carácter orientativo. El usuario es responsable de revisar, editar y verificar toda información antes de publicarla. La IA puede cometer errores o generar información inexacta.
